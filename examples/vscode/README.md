# VS Code in Browser Example

Run VS Code (code-server) in your browser, managed by mcp-sidecar.

## Prerequisites

```bash
brew install code-server
# or: npm install -g code-server
```

## Usage

```bash
cd examples/vscode
claude
```

Then in Claude Code:
```
> start the vscode process
```

VS Code will open in a Playwright browser at http://localhost:8443
