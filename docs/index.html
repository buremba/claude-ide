Keep Claude Code interactive
while it works.

Termos keeps you engaged while Claude runs. AskUserQuestion blocks today; Termos streams questions asynchronously, keeps Claude moving, and surfaces plans, proof, and tests so you decide the next step without losing context.

1
Install the CLI and the Claude plugin. 
npm install -g @termosdev/cli
claude plugins install termos
2
Run /termos:start in Claude to start the session.
3
Claude streams interactions with termos up and triggers termos run for questions. Replies wake Claude immediately if it is waiting, or are injected into context when the run finishes.

Async questions

Claude can ask mid-run without blocking, and you answer when ready.

Proof while it runs

Plans, diffs, and tests show up as evidence before the next step.

Keep context

Short check-ins prevent drift so you stay synced with the agent.
